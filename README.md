## Features

-add new task

-mark task as important

-set task deadline

-select task color

-mark task as finished

-delete task

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
